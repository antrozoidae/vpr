# SchoolExamPrinter v2.0

Автоматизация печати экзаменационных материалов для школ.

## Описание

Приложение для автоматизации процесса подготовки и печати экзаменационных материалов:
- Матрица классов (параллели × буквы)
- Матрица предметов с выбором классов и режимом "2 части"
- Управление локальными принтерами Windows
- Автоматический расчёт тиражей по вариантам
- Распределение нагрузки между принтерами
- Склейка многосоставных PDF файлов
- Тихая печать через SumatraPDF CLI с поддержкой дуплекса
- Возможность отмены печати с очисткой очереди

## Требования

- **ОС:** Windows 10/11 (x64)
- **Python:** 3.10+
- **SumatraPDF:** установленная или portable версия

## Установка

### 1. Установка Python

Скачайте и установите Python 3.10+ с [python.org](https://www.python.org/downloads/)

### 2. Установка зависимостей

```bash
cd school_exam_printer
pip install -r requirements.txt
```

### 3. Установка SumatraPDF

Скачайте SumatraPDF с [официального сайта](https://www.sumatrapdfreader.org/free-pdf-reader.html)

Или используйте portable версию, распаковав в удобную папку.

## Запуск

```bash
python main.py
```

## Использование

### 1. Вкладка "Классы"
Заполните матрицу классов: укажите количество учеников для каждой параллели (5-11 классы) и буквы (А, Б, В...).

### 2. Вкладка "Предметы"
- Добавьте предметы
- Для каждого предмета отметьте классы, которые будут сдавать
- При необходимости включите режим "2 части" для параллелей

### 3. Вкладка "Принтеры"
- Обновите список принтеров
- Выберите активные принтеры
- Настройте режим дуплекса для каждого

### 4. Вкладка "Задания"
Укажите пути к PDF файлам для каждого предмета, параллели и варианта.

### 5. Вкладка "Расчёт и Печать"
- Нажмите "Рассчитать" для просмотра распределения
- Нажмите "Печать" для начала печати
- Используйте "Отмена" для остановки процесса

## Сборка EXE

Для создания standalone исполняемого файла:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name SchoolExamPrinter main.py
```

## Логирование

Логи сохраняются в `%TEMP%/SchoolExamPrinter/print_log.txt`

## Отмена печати

При нажатии кнопки "Отмена":
1. Устанавливается внутренний флаг остановки
2. Прекращается отправка новых заданий
3. Очищается очередь спулера Windows (фильтрация по имени задания)

## Примечания

- При нечётном числе учеников: Вариант 1 = ceil(N/2), Вариант 2 = floor(N/2)
- Временные PDF создаются в `%TEMP%/SchoolExamPrinter/` и удаляются после печати
- Для работы со спулером Windows требуется pywin32 (устанавливается отдельно)
