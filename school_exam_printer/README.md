# SchoolExamPrinter

Автоматизация печати экзаменационных материалов для школ.

## Описание

Десктопное приложение на Python для Windows, которое позволяет:
- Настраивать классы и количество учеников
- Создавать предметы и привязывать их к классам
- Выбирать активные локальные принтеры
- Привязывать PDF-файлы заданий к вариантам
- Автоматически рассчитывать тиражи по вариантам
- Распределять задания между принтерами
- Выполнять тихую печать через SumatraPDF CLI с поддержкой дуплекса

## Требования

- **ОС:** Windows 10/11 (x64)
- **Python:** 3.10+
- **SumatraPDF:** установленная или portable-версия

## Установка

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Установка SumatraPDF

Скачайте и установите SumatraPDF с официального сайта:
https://www.sumatrapdfreader.org/free-pdf-reader.html

Или используйте portable-версию.

## Запуск

```bash
python main.py
```

## Использование

### 1. Вкладка "Классы"
- Добавьте классы: укажите параллель (1-11), букву и количество учеников
- Пример: 9А, 25 учеников

### 2. Вкладка "Предметы"
- Создайте предмет (например, "История")
- Выберите целевые классы из списка
- Отметьте флаг "Задание состоит из двух частей", если нужно

### 3. Вкладка "Принтеры"
- Нажмите "Обновить список принтеров"
- Отметьте чекбоксами активные принтеры

### 4. Вкладка "Задания"
- Выберите предмет из выпадающего списка
- Для каждого варианта (1 и 2) укажите пути к PDF файлам
- Если предмет состоит из двух частей, укажите оба файла

### 5. Вкладка "Расчёт и Печать"
- Выберите режим дуплекса: Нет / Длинная сторона / Короткая сторона
- Укажите путь к SumatraPDF.exe
- Нажмите "Рассчитать" для просмотра таблицы заданий
- Нажмите "Печать" для запуска
- При необходимости нажмите "Отмена" для остановки печати

## Конфигурация

Приложение автоматически сохраняет конфигурацию в `config.json` при закрытии.
Также доступны ручные команды "Сохранить" и "Загрузить" в меню "Файл".

### Формат JSON

```json
{
  "classes": [
    {"parallel": 9, "letter": "А", "students": 25}
  ],
  "subjects": [
    {
      "name": "История",
      "target_classes": ["9А"],
      "has_two_parts": true,
      "files": {
        "variant_1": {"part_1": "path/to/file.pdf", "part_2": "path/to/file2.pdf"},
        "variant_2": {"part_1": "path/to/file3.pdf", "part_2": "path/to/file4.pdf"}
      }
    }
  ],
  "printers": [
    {"name": "HP LaserJet Pro M404", "enabled": true}
  ],
  "settings": {
    "duplex_mode": "none",
    "sumatra_path": "C:/Program Files/SumatraPDF/SumatraPDF.exe"
  }
}
```

## Логирование

Все события печати записываются в файл `print_log.txt`:
- Время события
- Предмет и вариант
- Принтер
- Количество копий
- Статус и код возврата

## Сборка в EXE

Для создания standalone-приложения используйте PyInstaller:

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name SchoolExamPrinter main.py
```

## Особенности

- **Нечётное количество учеников:** Вариант 1 получает ceil(N/2), Вариант 2 — floor(N/2)
- **Распределение по принтерам:** Равномерное, остаток распределяется на первые N принтеров
- **Временные файлы:** Склеенные PDF создаются в `%TEMP%/SchoolExamPrinter/` и удаляются после печати
- **Отмена печати:** Остановка отправки новых заданий + очистка очереди спулера Windows
